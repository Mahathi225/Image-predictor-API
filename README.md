# Image Prediction API
## Project Overview
This project is a Flask-based web application that allows users to upload an image through a web interface. After uploading, the application processes the image and displays a prediction result. It demonstrates the basic implementation of an image prediction API using Python and Flask.
## Features
- Upload images from the browser
- Simple and user-friendly interface
- Displays image prediction results
- Lightweight and easy to understand
- Suitable for beginners learning Flask APIs
## Technologies Used
- Python
- Flask
- HTML
- Requests
- NumPy
- Pillow
